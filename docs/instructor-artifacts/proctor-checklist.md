# Proctor completion checklist

Use this page during a live cohort to track which hands-on labs each learner has finished (or is blocked on). It is **HTML-only in v1.0** — print from the browser or copy the table into a spreadsheet if you need a shared tracker. PDF export for facilitator artifacts is a **Phase 2** enhancement, not required for v1.0.

## How to use

1. **Print-and-check:** Open this page in the instructor site (`make serve-instructor` or the built `site-instructor/`), use the browser print dialog, and mark cells on paper.
2. **Spreadsheet:** Select the table, copy, and paste into Excel or Google Sheets; add rows for additional learners as needed.
3. **Per-lab column:** Check or initial a cell when that learner has completed the lab to your satisfaction (including any verification steps you require).
