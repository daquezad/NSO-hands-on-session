# Facilitator-only material

This section is **not** included in **learner** site builds (`make build-learner`, default `mkdocs serve`). It **is** included when you run **`make build-instructor`** (`INSTRUCTOR=1`).

Use it for facilitator checklists, timing aids, and other content that must not ship to learners. Long-form notes can also live in **`docs/NN-*-instructor-notes.md`** next to a lab chapter (same exclusion rules).

## Artifacts

- **[Proctor completion checklist](proctor-checklist.md)** — per-learner / per-lab grid for cohort tracking (Story 5.4, FR29).
- **[Per-chapter timing sheet](timing-sheet.md)** — durations and running totals from chapter frontmatter (Story 5.5, FR30).
