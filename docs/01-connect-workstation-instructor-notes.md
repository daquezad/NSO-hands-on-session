# Lab 1 — instructor companion notes

This file sits beside `01-connect-workstation.md`. It is **omitted** from learner builds and **included** in `site-instructor/` (Story 5.1).

## Suggested pacing

- If the class already has VPN and browser access warm, Lab 1 often finishes in **~10 minutes**.
- If Remote Access is flaky, budget **15–20 minutes** before starting Lab 2.

## Answers / staging

- **Black VM console:** Have learners refresh the Remote Access page; avoid nested RDP confusion (one console session at a time).
- **VS Code missing:** Activities menu → search “Code” or use a plain terminal until the shared image is fixed.

Keep short **inline** instructor-gated tips in the learner chapter for at-a-glance facilitation; use **this companion** for prose that would clutter the learner page or break the five-line inline guideline (lint rule 8).
